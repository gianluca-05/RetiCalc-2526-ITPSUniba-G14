Nitti Gianluca, Marangi Gabriel, Palmisano Lorenzo

La struttura della consegna è organizzata in due cartelle principali: TCP e UDP.  
All’interno di ciascuna sono presenti due cartelle conteneti i file .c  relativi al client e al server, con la seguente nomenclatura:

Consegna TCP:
- client-TCP_G14
  Contiene il sorgente `client-TCP_G14.c`
- server-TCP_G14
  Contiene il sorgente `server-TCP_G14.c`

Consegna UDP:
- client-UDP_G14
  Contiene il sorgente `client-UDP_G14.c`
- server-UDP_G14
  Contiene il sorgente `server-UDP_G14.c`

Il sistema operativo utilizzato per la realizzazione e i test delle applicazioni è: Macos
